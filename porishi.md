# PinterestApp

プライバシーポリシー（Privacy Policy）

本プライバシーポリシーは、当サービス（以下「本サービス」）がユーザーから取得する情報、その利用目的、第三者提供について説明するものです。本サービスをご利用いただく前に、本ポリシーをよくお読みください。

1. 収集する情報について

本サービスでは、以下の情報を取得する場合があります。

ユーザーが Pinterest API を通じて許可した情報
（Pinterest アカウント情報、公開プロフィール、ボード情報、画像データ など）

投稿のタイトル、説明文、画像など、本サービス上でユーザーが入力したデータ

サービス利用時に自動的に送信される技術情報
（IP アドレス、ブラウザ情報、端末情報、アクセス日時 など）

本サービスは、ユーザーが許可しない情報を Pinterest から取得することはありません。

2. 情報の利用目的

収集した情報は以下の目的のために利用します。

Pinterest への投稿自動化や関連機能の提供

ユーザー操作の簡略化および利便性向上

不具合の調査やサービス改善

必要に応じたユーザーサポートの提供

3. 第三者への提供について

本サービスは、以下の場合を除き、取得した情報を第三者に提供することはありません。

ユーザーが Pinterest API 利用のために許可した範囲で、Pinterest に送信される場合

法令に基づく場合

システム運用上必要な外部サービスを利用する場合（例：アクセス解析ツール）

4. クッキー等の利用

本サービスは、利用状況の分析や機能改善のためにクッキー（Cookie）を使用する場合があります。
クッキーにより個人が特定される情報を直接取得することはありません。

5. 情報の保存期間

ユーザーのデータは、本サービス提供に必要な期間のみ保持し、目的達成後またはユーザーからの削除依頼に応じて適切に削除します。

6. 安全管理措置

取得した情報は、不正アクセス、漏洩、改ざん等を防止するため、適切な管理措置のもとで運用されます。

7. 未成年の利用について

本サービスは、13 歳未満の児童による利用を意図していません。

8. プライバシーポリシーの変更

法令変更やサービス内容の更新に伴い、本ポリシーを改定することがあります。
変更後のポリシーは、本ページに掲載した時点から効力を有します。
